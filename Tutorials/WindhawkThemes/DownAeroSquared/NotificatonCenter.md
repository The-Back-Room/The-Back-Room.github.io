## DownAero Squared Notification Center Setup
Instructions for setting up the DownAero Squared theme's Notification Center using Windhawk.

### Requirements
- [Windhawk](https://windhawk.net/)
- [Windows 11 Notification Center Styler](https://windhawk.net/mods/windows-11-notification-center-styler)

### Instructions

1. If you haven't already, download and install Windhawk from the [official website](https://windhawk.net/).
2. Download and install the [Windows 11 Notification Center Styler](https://windhawk.net/mods/windows-11-notification-center-styler) mod through Windhawk.
3. Copy and paste the code below into `Windhawk >> Windows 11 Notification Center Styler >> Advanced >> Mod Settings` and click `Save`:
