## Naruto Hentai

A NSFW theme pack for Windows 10 and 11 inspired by the Naruto anime and manga series.

![Preview](https://gitlab.com/the-back-room/deskthemepacks/nsfw/naruto-hentai/-/raw/main/Extras/Preview.bmp)

{: .note }
> {: .opaque }
> The screenshot displays the theme being used alongside my DownAero Squared Windhawk theme. You can find instructions on how to obtain and set that up [here](https://the-back-room.info/WindowsThemes/WindhawkThemes/DownAeroSquared).

### Features

- Multiple Wallpapers
- Custom Cursors
- Custom Sounds
- Supports 4k Resolutions

### Instructions

- Download and extract the archive via the download button below.
- Double-Click the `.deskthemepack` file included to install the theme.
