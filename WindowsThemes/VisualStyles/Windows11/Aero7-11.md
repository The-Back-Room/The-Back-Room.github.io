### Features

- Multiple Styles
- Custom Cursors
- Custom Sounds

### Changing Styles
Due to the number of themse included in this collection, the styles are not seperated into their own different theme files.

To change the style simply open the desired `.theme` file in a text editor and change the `Path` value in `[VisualStyles]` to one of the following:

- `Path=%SYSTEMROOT%\Resources\Themes\Aero7-11\styles\Windows 7\aero7-11 round.msstyles`
- `Path=%SYSTEMROOT%\Resources\Themes\Aero7-11\styles\Windows 7\aero7-11 square.msstyles`
- `Path=%SYSTEMROOT%\Resources\Themes\Aero7-11\styles\Windows 7\aero7-11 round small-title.msstyles`
- `Path=%SYSTEMROOT%\Resources\Themes\Aero7-11\styles\Windows 7\aero7-11 square small-title.msstyles`
- `Path=%SYSTEMROOT%\Resources\Themes\Aero7-11\styles\Windows Vista\aeroVista-11 round.msstyles`
